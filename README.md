# 寒假成果報告

這是一份單一 HTML 檔案，包含整個網站的內容（CSS、圖片、影片），無需任何外部資源。

## 使用方式

1. **下載或克隆整個資料夾**
   
   - 如果老師不熟悉命令列，可以直接在瀏覽器中點選 GitHub 頁面的「Code」按鈕，選擇 "Download ZIP" 下載壓縮包，解壓後即可。
   - 若會使用命令提示字元（Terminal、命令列視窗），請在電腦上打開一個命令列視窗，然後鍵入：
     ```bash
     git clone https://github.com/andychou978/Andy-s-winter-report-2026.git
     cd Andy-s-winter-report-2026
     ```
     命令列視窗就是像黑底白字的程式，或 Windows 的 PowerShell/命令提示字元。
     
     **如何開啟 Windows 命令提示字元 (CMD)：**
     1. 在鍵盤上按下`Win`鍵（有 Microsoft 標誌的鍵）
     2. 輸入「cmd」或「命令提示字元」
     3. 按下 `Enter`，就會出現黑底的命令視窗。
     4. （也可以在「開始」菜單中找到「命令提示字元」或「PowerShell」，右鍵選擇開啟。）

2. **打開檔案**
   - 在資料夾裡找到名為 `寒假成果報告.html` 的檔案（通常圖示是一個網頁樣式的圖標）。點兩下它就會在預設瀏覽器中打開。
   - （下方示意圖）
     
     ![HTML 檔案圖示範例](screenshot.png)
     
   - 或使用瀏覽器開啟 `file:///` 路徑。   - **不需要安裝 Python**：只要有瀏覽器就可以直接開啟，執行伺服器只是給想用網頁形式瀏覽整個資料夾的人用，老師可以忽略這一步。
3. **推薦（不必要）**
   若要確保所有相對連結正常，可在本資料夾執行簡單的 HTTP 伺服器：
   ```bash
   # 例如使用 Python
   cd path/to/Andy-s-winter-report-2026
   python -m http.server 8000
   # 然後在瀏覽器訪問 http://localhost:8000/
   ```

4. **分享**
   將 `寒假成果報告.html` 直接發送給老師或同學即可，該檔案中已包含所有必要內容。
