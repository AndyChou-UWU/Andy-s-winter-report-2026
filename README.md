# 寒假成果報告

這是一份單一 HTML 檔案，包含整個網站的內容（CSS、圖片、影片），無需任何外部資源。

## 使用方式

1. **下載或克隆整個資料夾**
   ```bash
   git clone https://github.com/andychou978/Andy-s-winter-report-2026.git
   cd Andy-s-winter-report-2026
   ```
   或直接從 GitHub 下載壓縮包並解壓。

2. **打開檔案**
   - 直接在檔案管理器中雙擊 `寒假成果報告.html`。
   - 或使用瀏覽器開啟 `file:///` 路徑。

3. **推薦（不必要）**
   若要確保所有相對連結正常，可在本資料夾執行簡單的 HTTP 伺服器：
   ```bash
   # 例如使用 Python
   cd path/to/Andy-s-winter-report-2026
   python -m http.server 8000
   # 然後在瀏覽器訪問 http://localhost:8000/
   ```

4. **分享**
   將 `寒假成果報告.html` 直接發送給老師或同學即可，該檔案中已包含所有必要內容。
